About the Experiment

In this experiment, I have implemented a login page using bootstrap. Here, the email id input field is of type email, which gives error if text which is not an email id is entered. The password input field is of type password, in order to display the passwords entered in form of stars(***), to ensure security from people standing nearby.

Code Snippet

HTML 
    <div class="wrapper">
        <form class="form-signin">
            <h2 class="form-signin-heading">Please login</h2>
            <input type="email" class="form-control" name="username" placeholder="Email Address" required="" autofocus="" />
            <input type="password" class="form-control" name="password" placeholder="Password" required="" />
            <label class="checkbox">
                <input type="checkbox" value="remember-me" id="rememberMe" name="rememberMe"> Remember me
            </label>
            <button class="btn btn-lg btn-primary btn-block" type="submit">Login</button>
        </form>
    </div>
