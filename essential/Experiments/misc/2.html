Images SlideShow using JS



About the Experiment

In this experiment, on clicking the button, a slideshow of Images starts.
Image on the frame changes after every two seconds. It goes on in a loop.

Code Snippet

Javascript: 
    var mySlideShow = document.getElementById('mySlideShow');
    var myPics = ['a.jpg', 'b.jpg', 'c.jpg', 'd.jpg', 'e.jpg', 'f.jpg', 'g.jpg', 'h.jpg', 'i.jpg', 'j.jpg', 'k.jpg', 'l.jpg'];
    var PicCount = myPics.length;
    var i = 0;
    function loop() {
        if (i > (PicCount - 1)) {
            i = 0;
        }
    mySlideShow.innerHTML = '';
    i++;
    loopTimer = setTimeout('loop()', 1200);
    }
HTML : 
    <div id="myFrame">
    <div id="mySlideShow">
    <input type="button" class="btn btn-success" value="Click to start slide show" onclick="loop()"><br /><br />
    </div>
    </div>
