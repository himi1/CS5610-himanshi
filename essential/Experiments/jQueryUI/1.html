About the Experiment

In this experiment, I have experimented on jQeueryUI. I implemented the following functionalities:

  • Draggable
  • Resizable
  • Selectable
  • Sortable
  • Accordion
I have also implemented auto-complete of Programming Languages(from a given list) using jQuery.

Code Snippet

JS 
$(function () {
    $(".draggable").draggable();
    $("#resizable").resizable();
    $("#selectable").selectable();
    $("#sortable").sortable();
    $("#sortable").disableSelection();
    $("#accordion").accordion();
});
