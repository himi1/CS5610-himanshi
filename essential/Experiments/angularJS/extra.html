Angular JS httprequest

Reading of static data from another web page

  • {{ x.Name + ', ' + x.Country }}

About the Experiment

This experiment is performed to learn to make a httprequest using AngularJS
AngularJS will invoke customersController with a $scope and $http object. $scope is the application object (the owner of application variables and functions). $http is an XMLHttpRequest object for requesting external data. $http.get() reads static JSON data from http://www.w3schools.com/website/Customers_JSON.php. If success, the controller creates a property (names) in the scope, with JSON data from the server.

Code Snippet

JS 
        <script>
        function customersController($scope, $http) {
        $http.get("http://www.w3schools.com//website/Customers_JSON.php")
        .success(function (response) { $scope.names = response; });
        }
        </script>
HTML 
        <div ng-app="" ng-controller="customersController">
        <ul>
        <li ng-repeat="x in names">
        {{ x.Name + ', ' + x.Country }}
        </li>
        </ul>
        </div>
