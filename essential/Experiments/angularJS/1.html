About the Experiment

I did this experiment to learn AngularJS basics.
AngularJS starts automatically when the web page has loaded. The ng-app directive tells AngularJS that the <div> element is the "owner" of an AngularJS application.
The ng-model directive binds the value of the input field to the application variable name.
The ng-bind directive binds the innerHTML of the <p> element to the application variable name.

Code Snippet

JS 
        <script>
       function personController($scope) {
        $scope.firstName = "First name";
        $scope.lastName = "Last name";
        };
        </script>
HTML 
        <div ng-app="" ng-init="FirstName='Himanshi'" ng-controller="personController">
        <p>The name is <span ng-bind="FirstName"></span></p>
        <p>My first expression: {{ 10 + 10 }}</p>
        First Name: <input type="text" ng-model="firstName"><br>
        Last Name: <input type="text" ng-model="lastName"><br>
        <br>
        Full Name: {{firstName + " " + lastName}}
        </div>
