About the Experiment

An AngularJS Application is implemented in this experiment. It takes in the typed data, displays it on MyNote. It also returns the Number of characters left, that may be typed in MyNote.

Code Snippet

Javascript:
mynoteapp.js 
            var app = angular.module("myNoteApp", []);
mynotecontrol.js 
            app.controller("myNoteCtrl", function ($scope) {
            $scope.message = "";
            $scope.left = function () { return 500 - $scope.message.length; };
            $scope.clear = function () { $scope.message = ""; };
            $scope.save = function () { alert("Note Saved"); };
            });
HTML 
            <body ng-app="myNoteApp" ng-controller="myNoteCtrl">
            <textarea ng-model="message" cols="50" rows="10"></textarea>
            <p> <button ng-click="save()">Save</button>
            <button ng-click="clear()">Clear</button>
            </p>
            <p>Number of characters left: <span ng-bind="left()"></span></p>
