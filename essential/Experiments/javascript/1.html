Prompt using JavaScript

About the Experiment

In this experiment, I implemented Prompts on an HTML page using JavaScript.

Code Snippet

JS 
var answer = prompt("Type your name here: ");
alert("Hello " + answer + "!");
answer1 = prompt('What is your name again');
if (answer1 == answer) {
    alert("Correct!");
} else if (answer1 != answer) {
    alert("Incorrect! Thou shalt not pass!");
}
answer2 = prompt("What's your facebook password?");
if (answer2 == "fb1234") {
    alert("Correct!");
} else if (answer2 != answer) {
    alert("Incorrect Password!");
}
